삼성 클라우드의 새로운 약관이 적용되었습니다
안녕하세요,
Samsung Cloud 서비스를 이용해 주시는 고객 여러분께 진심으로 감사드립니다.
2018년 9월 30일부로 개정된 Samsung Cloud 약관이 적용 되었습니다.

Samsung Cloud에 보관된 데이터가 12개월 넘게 전혀 사용되지 않으면, 전체 데이터는 파기됩니다. 또한 Samsung Cloud의 특정 콘텐츠 카테고리의 데이터가 12개월 넘게 전혀 사용되지 않은 경우에도 마찬가지로 해당 콘텐츠 카테고리의 데이터가 파기됩니다.

사용자는 데이터가 파기되기 30일 전부터 고지를 받게 되며, 이 경우 Samsung Cloud에 접속하여 데이터에 접근하기만 하더라도 '데이터의 사용'이 되어 데이터를 보존할 수 있습니다.

[약관의 해당 조항]
3.9 사용자가 Samsung Cloud서비스를 전부 또는 콘텐츠 카테고리 별로 사용하지 않는 기간이 12개월을 초과하는 경우, 삼성은 30일전에 고객에게 사전 고지 후 개인정보 보호와 서비스 자원 확보를 위해 사용자의 콘텐츠 및 정보를 전부 또는 콘텐츠 카테고리별로 복구가 불가능한 방법으로 파기합니다. 여기서 사용이란 사용자의 콘텐츠 및 정보의 접근, 업로드, 다운로드를 의미합니다. 사용자는 파기되기 전에 사용자 자신의 책임으로 콘텐츠 및 정보를 백업 또는 저장하여야 하며, 사용자가 백업 또는 저장하지 않아 발생하는 어떠한 손실에 대해서도 삼성은 책임지지 않습니다.
본 약관에 대한 자세한 사항은 Samsung account 웹사이트(https://account.samsung.com/membership/terms)에서 확인하실 수 있습니다.

Samsung Cloud 서비스를 이용해 주시는 고객 여러분께 다시 한 번 감사드립니다.
